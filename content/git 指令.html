git 指令

在 c:tmp 目錄中建立一個test_repo 目錄

cd ..  指令是返回前個動作

打 D: 就會進入D槽 打 E: 就會進入E槽

python wsgi.py 動態擷取系統

git status 檢視修改

git add . 將檔案納入追蹤

git pull 靜動態整合

git commit -m "initial"將檔案提交

git push 上傳檔案
